01. Sobre mí

Soy desarrollador web junior de Madrid, recién titulado en el ciclo superior de Desarrollo de Aplicaciones Web. Me gusta entender los proyectos de punta a punta: no me conformo con que el código funcione en mi máquina, sino que llegue desplegado, seguro y mantenible a producción.

He desarrollado desde cero la plataforma web de una tienda de muebles real —catálogo público, API REST con autenticación y panel de administración— y una web corporativa para una empresa de ingeniería frigorífica. Trabajo a diario con Linux, Git y Docker.

Busco un equipo donde seguir creciendo como desarrollador full stack, aportar desde el primer día y aprender de gente con más experiencia.
